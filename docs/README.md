### 🚀 快速开始

如果您是第一次使用，建议按以下步骤开始：

1. **安装程序**
   ```bash
   git clone https://github.com/Tavre/lchliebedich.git
   cd lchliebedich
   pip install -r requirements.txt
   ```

2. **启动程序**
   ```bash
   python main.py
   ```

3. **配置OneBot连接**
   - 打开"工具" → "配置管理"
   - 填入OneBot服务器信息
   - 点击"测试连接"

4. **导入词库**
   - 点击"文件" → "导入词库"
   - 选择词库文件

5. **开始使用**
   - 查看实时消息日志
   - 监控统计信息

### 📋 主要功能

- **词库管理**: 导入、编辑、导出词库文件
- **伪代码支持**: 丰富的伪代码功能
- **实时监控**: 消息日志和连接状态监控
- **统计分析**: 详细的使用统计和图表
- **现代化界面**: 基于PyQt5的美观界面

### 🔧 系统要求

- Python 3.7+
- PyQt5 5.12+
- 100MB+ 可用磁盘空间
- 网络连接（OneBot通信）
